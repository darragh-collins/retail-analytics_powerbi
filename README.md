# Retail Analytics Dashboard Project

## Overview
This project is an end-to-end retail analytics solution built in Power BI.

The goal of the project was to simulate a real business reporting environment using:
- Star schema data modelling
- Power Query transformations
- DAX measures
- Business-focused KPI reporting
- Interactive dashboard design

The project focuses on analysing:
- Sales performance
- Product performance
- Inventory management
- Returns analysis
